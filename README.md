# Spotify Clone

## Getting Started

- frontend: "npm start"
- backend: CD into "server" folder, "npm run DevStart"

## Tool & Technologies

- Spotify for Developers
- Bootstrap
- React Bootstrap
- [Spotify-Web-Api-Node](https://github.com/thelinmichael/spotify-web-api-node)
- Express
- Axios
- CORS
- Body parser
- React Spotify Web playback
- Lyrics finder
